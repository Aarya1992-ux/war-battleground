🎮 WAR BATTLEGROUND - QUICK START (30 MIN)
============================================

✅ STEP 1: Database (5 min)
  □ Go to supabase.com → Sign up
  □ Create project: "war-battleground"
  □ SQL Editor → Copy-paste server/db/schema.sql → RUN
  □ Settings → Database → Copy CONNECTION STRING
  
✅ STEP 2: Server (5 min)
  □ Open Command Prompt (NOT PowerShell!)
  □ cd Desktop\battleground-final\server
  □ Create .env file:
     PORT=8080
     DATABASE_URL=[YOUR_SUPABASE_URL]
     DATABASE_SSL=true
     JWT_SECRET=randomSecretHere123456
  □ npm install
  □ npm start
  → Should see: "listening on port 8080" ✅

✅ STEP 3: Client Setup (10 min)
  □ Open NEW Command Prompt
  □ cd Desktop\battleground-final\client
  □ npm install
  □ npm install vite-plugin-pwa workbox-build
  □ npm run dev
  → Should see: "http://localhost:5173" ✅

✅ STEP 4: CRITICAL Manual Updates (5 min) ⚠️

✅ STEP 5: Test Locally
  □ Refresh browser (http://localhost:5173)
  □ Sign up with invite code (get from DB):
     SELECT code FROM invites LIMIT 1;
  □ Should see: Cosmetics Shop, Battle Pass, Campaign buttons
  □ Click buttons → Modals should open

✅ STEP 6: Deploy (10 min)
  □ Create GitHub repo:
     git init
     git add .
     git commit -m "War Battleground"
     git branch -M main
     git remote add origin https://github.com/YOUR_USERNAME/war-battleground.git
     git push -u origin main
  
  □ Deploy Server (Railway):
     railway.app → New Project → GitHub → war-battleground
     Root: server
     Env vars:
       DATABASE_URL=[Supabase URL]
       JWT_SECRET=randomKey
       DATABASE_SSL=true
     Copy deployed URL (e.g., war-battleground.railway.app)
  
  □ Deploy Client (Vercel):
     vercel.com → Import Project → war-battleground
     Root: client
     Env vars:
       VITE_WS_URL=wss://war-battleground.railway.app
       VITE_API_URL=https://war-battleground.railway.app/api
     Deploy → Get live URL
